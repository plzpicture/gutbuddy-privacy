오늘도오픈 개인정보 처리방침

시행일: 2026년 5월 17일

앱 소개

오늘도오픈은 가게 사장님이 말하기, 사진 스캔, 직접 입력으로 매출, 비용, 상품, 재고, 수익을 기록할 수 있도록 돕는 로컬 전용 iOS 장부 앱입니다.

수집하는 개인정보

앱 개발자는 사용자의 개인정보, 장부 데이터, 상품 데이터, 판매 데이터, 음성 내용, 영수증 사진, 재고 정보를 서버로 수집하지 않습니다.

기기 안에 저장되는 데이터

  • 수입/지출 거래
  • 상품명, 판매가, 원가, 재고, 단위
  • 상품별 판매 기록
  • 카테고리
  • 앱 설정

카메라, 마이크, 음성 인식 사용

앱은 사용자가 직접 선택한 경우에만 카메라, 마이크, 음성 인식을 사용합니다.

  • 카메라: 영수증이나 판매 메모를 읽어 장부 입력을 돕기 위해 사용합니다.
  • 마이크: 말로 장부, 상품, 판매 내용을 입력하기 위해 사용합니다.
  • 음성 인식: 말한 내용을 텍스트로 바꾸기 위해 사용합니다.

제3자 제공 및 추적

오늘도오픈은 광고 SDK, 추적 SDK, 분석 SDK를 포함하지 않습니다. 앱 개발자는 사용자의 데이터를 제3자에게 판매하거나 공유하지 않으며, 사용자를 추적하지 않습니다.

백업과 삭제

현재 앱 데이터는 사용자의 iPhone 기기 안에 저장됩니다. 앱을 삭제하거나 기기를 분실하면 데이터가 사라질 수 있습니다. 향후 백업/내보내기 기능이 추가되면 사용자가 직접 파일을 저장하거나 공유할 수 있습니다.

문의
